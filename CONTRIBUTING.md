## কিভাবে শুরু করবেন?

### Step 1: Fork করুন
```
GitHub → https://github.com/halima1971/-Lutera
"Fork" বাটন ক্লিক করুন
```

### Step 2: Clone করুন
```bash
git clone https://github.com/YOUR-USERNAME/-Lutera.git
cd -KhaLLas
```

### Step 3: Feature Branch তৈরি করুন
```bash
git checkout -b feature/your-feature-name
```

### Step 4: পরিবর্তন করুন
- Android Studio এ খুলুন
- কোড পরিবর্তন করুন
- Test করুন

### Step 5: Commit করুন
```bash
git add .
git commit -m "Add your feature description"
```

### Step 6: Push করুন
```bash
git push origin feature/your-feature-name
```

### Step 7: Pull Request তৈরি করুন
```
GitHub → Compare & pull request → Create Pull Request
বিবরণ লিখুন এবং সাবমিট করুন
```

## 🐛 বাগ রিপোর্ট

### বাগ খুঁজে পেলে:

1. [GitHub Issues](https://github.com/halima1971/-Lutera/issues) এ যান
2. "New Issue" ক্লিক করুন
3. বাগের বিস্তারিত বর্ণনা দিন:
   - **Title:** বাগের সংক্ষিপ্ত বিবরণ
   - **Description:** বিস্তারিত
   - **Steps to Reproduce:** কিভাবে reproduce করা যায়
   - **Expected vs Actual:** কি হওয়া উচিত vs কি হচ্ছে
   - **Screenshots:** যদি প্রাসঙ্গিক হয়
